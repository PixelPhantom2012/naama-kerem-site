# Required Images

Place the following image files in this folder before publishing the site:

| Filename        | Usage                          | Notes                                      |
|-----------------|--------------------------------|--------------------------------------------|
| `hero-bg.jpg`   | Hero section full-screen bg    | High-res JPG/WebP, both performers on stage |
| `about-photo.jpg` | About section portrait photo | Can be candid; both performers in frame    |

## Tips
- Preferred format: JPG or WebP (WebP gives smaller file size)
- Minimum resolution: 1920×1080 px for hero, 800×1000 px for about photo
- Compress images before uploading (use squoosh.app or tinypng.com)
- The `hero-bg.jpg` file is already referenced in `index.html`
